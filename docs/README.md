# alpdinamik

Alpdinamik - Next.js projesi

## Kurulum

Bağımlılıkları yükleyin:

```bash
npm install
```

## Geliştirme

Development server'ı başlatın:

```bash
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın.

## Build

Production build oluşturun:

```bash
npm run build
```

## Teknolojiler

- Next.js 14
- React 18
- Bootstrap 5
- Swiper
- WOW.js
